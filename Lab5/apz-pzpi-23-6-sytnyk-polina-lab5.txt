5.2 Короткий опис проєкту
Click&Rent — це web-система для онлайн-оренди одягу та аксесуарів, яка дозволяє користувачам переглядати каталог речей, оформлювати оренду та взаємодіяти із сервісом через зручний web-інтерфейс.
Проєкт реалізований як клієнт-серверна система. Серверна частина розроблена на платформі ASP.NET Core MVC та відповідає за обробку бізнес-логіки, роботу з базою даних, авторизацію користувачів, керування речами, орендами та адміністративними функціями. Для зберігання даних використовується Microsoft SQL Server.
Клієнтська частина представлена web-застосунком, який дозволяє користувачам переглядати каталог доступних речей, переходити між категоріями товарів, виконувати авторизацію та працювати з функціоналом оренди.
Для адміністраторів реалізовано окрему панель керування. Вона дозволяє виконувати управління користувачами системи, речами та орендами, а також створювати резервні копії даних.
У процесі виконання лабораторних робіт було також підготовлено систему до масштабування. Для цього використовувалися Docker, Docker Compose, балансувальник навантаження Nginx та інструмент навантажувального тестування Locust.
Таким чином, Click&Rent демонструє повний цикл роботи сучасного web-застосунку: від взаємодії користувача з каталогом речей до обробки запитів серверною частиною, збереження інформації в базі даних та адміністрування системи.
5.3 Заплановані та виконані функції
У межах проєкту було заплановано реалізувати основні компоненти системи онлайн-оренди одягу та аксесуарів: серверну частину, базу даних, web-інтерфейс користувача, систему авторизації, механізм оренди речей та адміністративну панель.
Основні заплановані функції були реалізовані. У системі працює авторизація користувачів, перегляд каталогу речей, поділ каталогу на категорії, управління речами, управління користувачами та управління орендами через адміністративну панель.
Також було реалізовано серверну частину на ASP.NET Core MVC, структуру бази даних Microsoft SQL Server, рольову модель доступу та механізм резервного копіювання даних. Для демонстрації масштабування системи були додані Docker, Docker Compose та конфігурація балансувальника навантаження Nginx.
Окремо було проведено навантажувальне тестування системи за допомогою Locust. Це дозволило оцінити поведінку серверної частини під навантаженням та проаналізувати потенційні вузькі місця системи.
Не всі можливості були доведені до повноцінного рівня. У поточній версії відсутня інтеграція з платіжними системами, система сповіщень та мобільний застосунок не дороблений. У майбутньому можливе розширення функціоналу за рахунок онлайн-оплати, push-повідомлень, мобільного клієнта та хмарного розгортання системи.
Отже, у межах роботи було реалізовано основний функціонал проєкту Click&Rent. Система вже демонструє ключову логіку продукту: користувач переглядає каталог, обирає речі для оренди, взаємодіє з системою через web-інтерфейс, а адміністратор керує користувачами, речами та орендами через окрему панель керування.
